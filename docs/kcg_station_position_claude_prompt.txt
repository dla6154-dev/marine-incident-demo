Important constraints:
- this endpoint is XML-only in live testing
- `type=json` and `resultType=json` did not switch it to JSON
- success response was:
  - `resultCode = 40`
  - `resultMsg = NON_ERROR`
- do not assume `00` is the only success code for this endpoint
- the endpoint appears to ignore paging in live tests
- use one-shot fetch:
  - `rowsCount=100000`
  - `startPage=1`
- this dataset only provides:
  - station name
  - longitude
  - latitude
- it does not provide address or phone number

1. create `kcg_stations.py`

3. parse XML records from `body > content`

5. expose a backend route such as `/api/kcg-stations`
6. optionally compute nearest station from the current accident coordinate
7. optionally render map markers
